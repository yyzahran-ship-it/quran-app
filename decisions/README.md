# Decisions

When Claude or Yazan makes a significant architectural or product decision,
log it here as `NNN-short-topic.md`.

Format:
```
# Decision: <topic>

Date: YYYY-MM-DD
Status: proposed | accepted | superseded by NNN

## Context
What problem is this solving?

## Options considered
1. Option A — pros, cons
2. Option B — pros, cons

## Decision
What we chose and why (one paragraph).

## Consequences
What this means going forward.
```

This is how the project keeps a memory of *why* things are the way they are,
which matters when someone (Claude or human) comes back months later.
